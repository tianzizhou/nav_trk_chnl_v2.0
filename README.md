# 多模态融合反无人机探测系统技术方案

## 项目概述

针对低慢小（LSS）无人机目标在超低空、近距、蛙跳机动、城市复杂背景等
场景下的探测难题，提出 **"雷达+光电+声学阵列+无线电侦测"四模态融合**
探测方案。

## 核心指标

| 指标 | 要求 | 预估性能 |
|------|------|---------|
| 目标速度 | ≤ 50 m/s | ✅ |
| 飞行高度 | 10 ~ 300 m | ✅ |
| 探测距离 | ≥ 3000 m | ~95% Pd @ 3000m |
| 定位精度 | CEP ≤ 5m @ 1km | ~1.2m |
| 系统成本 | ≤ 100万/套 | ~100万 |
| 安装平台 | 猛士车载 | ✅ |
