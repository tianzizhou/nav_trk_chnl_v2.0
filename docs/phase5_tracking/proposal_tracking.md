# 目标跟踪模块 — 技术方案文档

## 1. 概述与目标

多目标跟踪模块对检测到的目标进行状态估计、数据关联和轨迹管理，
在高速(>1000km/h)场景下保持跟踪连续性。

## 2. 候选方案对比

| 滤波算法 | 非线性处理 | 计算复杂度 | 适用性 |
|----------|-----------|-----------|--------|
| KF | 线性假设 | O(n²) | ★★★ |
| **EKF** | 一阶线性化 | O(n²) | **★★★★** |
| UKF | Sigma点 | O(n³) | ★★★★ |
| 粒子滤波 | 蒙特卡洛 | O(N·n) | ★★ |

| 关联算法 | 全局最优 | 复杂度 | 适用性 |
|----------|---------|--------|--------|
| GNN | 贪心 | O(N·M) | ★★★ |
| **Hungarian** | 全局最优 | O(N³) | **★★★★★** |
| JPDA | 概率 | O(exp) | ★★ |

## 3. 推荐方案

**9维EKF（匀加速模型）+ Hungarian匹配**
- 状态：[x,y,z, vx,vy,vz, ax,ay,az]
- 过程噪声：Singer加速度模型
- 测量噪声：距离自适应
- 预测驱动ROI：减少搜索范围

## 4. 高速目标挑战

| 挑战 | 解决方案 |
|------|----------|
| 帧间位移2.3m | 加大关联门限(50m) |
| 加速度不可预测 | Singer加速度噪声模型 |
| 短暂遮挡 | 5帧miss容忍 |
| 新目标出现 | 3帧确认后进入confirmed |
