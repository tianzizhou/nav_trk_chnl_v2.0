# 目标检测模块 — 技术方案文档

## 1. 概述与目标

目标检测模块负责在双目图像中定位无人机目标，输出检测框和置信度。

## 2. 候选方案对比

| 算法 | 参数量 | 推理延迟 | 小目标能力 | GPU需求 |
|------|--------|----------|------------|---------|
| YOLOv8n | 3.2M | ~3ms (TRT) | ★★★ | 需要 |
| YOLOv11n | 2.6M | ~2.5ms | ★★★★ | 需要 |
| RT-DETR | 20M+ | ~8ms | ★★★★ | 需要 |
| **模拟检测器** | 0 | ~0ms | ★★★★★ | **不需要** |

## 3. 推荐方案

**模拟检测器**（仿真阶段）：基于Ground Truth + 可配置噪声、漏检、虚警，
验证完整流水线的算法正确性。接口预留未来接入真实YOLO模型。

## 4. 检测概率模型

检测概率随距离衰减（Sigmoid函数）：
- 100m处：P_d ≈ 0.98
- 1500m处（半衰距离）：P_d ≈ 0.49
- 3000m处：P_d < 0.05

## 5. 噪声模型

- bbox位置噪声：σ = base_std × (distance / 100)
- 置信度：线性衰减 + 随机扰动
- 虚警：泊松分布，默认率 0.5%/帧
